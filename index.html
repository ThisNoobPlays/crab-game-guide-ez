Crab Game Introduction

Crab Game is a free multiplayer game where you compete in short, fun minigames to try and be the last one standing.

It runs on every major operating system: Windows, MacOS, and Linux.

One game normally starts with a maximum of 20 players, and in each game, some players are eliminated, and the rest move on to the next game. Eventually, there will be a final winner (or maybe 0 if fate befalls you).

The minigames include:

  • Tag - Avoid having the stick by tagging other people with it
  • Bomb Tag - Pass the bomb to someone else so you don't explode!
  • Hide and Seek - Run away from the seekers, or if you are a seeker, stab somebody before the game ends!
  • Race - Be the first to complete an obstacle course
  • Block Drop - Try not to get crushed by falling blocks!
  • And much, much more!

Install the Game

Crab Game is available for free on Steam. Click on the widget or link. You will need a Steam account to download the game, which is also free.

https://store.steampowered.com/app/1782210/Crab_Game/

Launching the Game

Now you are ready to play your first game! Start up Crab Game by searching "Crab Game" in your operating system's search bar.

If you can't find it there, open Steam, Click on "Library" at the top, click "Crab Game" on the left sidebar, and then click the green "Play" button.

Your first Crab Game

If there is a popup asking you something about Proximity Voice Chat, set it to Push to Talk.

Click on "Find Game" to bring up a list of active servers. A server is basically one "room" of players. Click "Join" on a server that is in the Lobby. That way, you won't have to wait for the previous game to finish.

When you load in, you spawn in the Lobby, which is where players wait while the game has not actually started yet.

Use WASD to move, Shift to run, and Spacebar to jump. confused?

When you are ready, go up to the middle of the lobby and find the red button. Press E to activate it, which will count you as "ready." When most of the players are ready, a countdown will appear at the top, and the game will start.

Read the instructions before each minigame to see how to play it.

Once you are a little familiar with how the game works, you can continue to the next sections to gain more information that is helpful for beginners. If you are experiencing performance issues, check out the Performance Optimization page.

How Servers Work

You might have noticed the "Start Game" button on the home screen. That is actually where you can create your own server.

What are Servers?

A server is a computer that is hosting a Crab Game room. When you join a specific server, your computer is connecting to the server computer and sending it lots of data such as your player position and where you're looking. The server computer takes in all of the data from all of the connected players and processes it before sending an output data back to your computer.

This is how you can see where other players are. Because that other player's computer is sending their position data to the server, which is then processing it and sending it back out to all the players so all the players can see where that player is.

Hosting your own Game

Instead of joining someone else's server, you can host your own server on your own computer and let other people join you. This is actually very easy and all you have to do is click the "Start Game" button on the home screen. This brings you to a menu where you can set the server name, the max players, the lobby type, and the minigames that should be allowed.

The lobby type is an important option:

Public
Your server will be listed in the "Find Game" section and anybody can join your server
Friends Only
Only your friends on Steam can see your game and join it
Lobby Code Only
Your server will not be listed in the "Find Game" section. You have to copy the lobby code and manually give it to people who you want to join your server. This is useful for creating a private lobby with your friends.

The rest of the options are pretty self-explanatory.
